FAQ – AsapCloud Platform

Encontre respostas rápidas sobre o funcionamento da AsapCloud Platform e como ela simplifica o provisionamento de infraestrutura na AWS.

1. O que é a AsapCloud Platform?

A AsapCloud Platform é uma Internal Developer Platform (IDP) criada para simplificar e automatizar o provisionamento de infraestrutura na AWS. Ela permite que usuários criem instâncias EC2 através de uma interface web ou via chatbot, mantendo padronização, segurança e eficiência.

2. Qual é o objetivo principal da plataforma?

O objetivo é reduzir a complexidade operacional e permitir que qualquer desenvolvedor, mesmo sem conhecimento avançado em AWS, consiga provisionar de forma rápida, segura e guiada uma máquina virtual EC2.

3. Preciso ter conhecimento em AWS para usar a AsapCloud Platform?

Não necessariamente. A plataforma abstrai quase toda a complexidade e apresenta apenas opções essenciais. O usuário pode criar uma EC2 sem acessar o console da AWS.

4. Como funciona a criação de uma EC2 pelo website?

O usuário acessa o site, preenche alguns parâmetros (tipo de instância, nome, região, chave SSH etc.) e o sistema dispara uma API em Python/Java que executa a criação da EC2 de forma automatizada.

5. O chatbot também consegue criar uma EC2?

Sim. O chatbot recebe um payload com os parâmetros necessários e envia esses dados para a API interna, que realiza todo o processo de criação da instância.

6. A AsapCloud Platform substitui ferramentas como Terraform ou CloudFormation?

Não. Ela pode utilizar essas tecnologias internamente, mas a proposta é criar uma camada de abstração e autoatendimento, facilitando o uso por pessoas que não querem (ou não precisam) lidar diretamente com IaC.
